# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - June 24, 2026

### Added

- TanStack Router file-based routing with optional static generation

### Changed

- Replaced **Webpack** with **Vite**
- Replaced **Jest** with **Vitest**
- Replaced **Bulma** with **Tailwind CSS**
- Migrated package manager from **npm** to **pnpm**
- Upgraded **ESLint** to v10
- Upgraded **TypeScript** to v6
- Upgraded **React** and other dependencies to latest stable versions
- Modernized TypeScript configuration
- Modernized ESLint and Prettier setup

### Fixed

- Imported Tailwind CSS in the root route

### Removed

- Cleaned up obsolete references

### Security

- Bumped `@babel/traverse` from 7.21.5 to 7.23.2
- Bumped `postcss` from 8.4.23 to 8.4.31
- Bumped `word-wrap` from 1.2.3 to 1.2.4
- Bumped `tough-cookie` from 4.1.2 to 4.1.3
